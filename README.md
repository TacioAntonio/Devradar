### Installation
1. Front-End
```sh
$ git clone https://github.com/TacioAntonio/Devradar
$ cd Devradar/web
$ yarn install
$ yarn start
```

2. Mobile
```sh
$ git clone https://github.com/TacioAntonio/Devradar
$ cd Devradar/mobile
$ yarn install
$ yarn start
```

3. Back-End
```sh
$ git clone https://github.com/TacioAntonio/Devradar
$ cd Devradar/backend
$ yarn install
$ yarn start
```
